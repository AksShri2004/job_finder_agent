# 🚀 Job Finder AI Agent Pipeline

An automated, multi-stage recruitment and career preparation pipeline built using the **Google ADK (Agents SDK)** and **Gemini 2.5 Flash**. This system transforms raw resume text into a comprehensive career strategy—covering structured data extraction, role matching, and interview coaching.

## 📌 Overview
The project utilizes a `SequentialAgent` named `Job_Finder_Agent`. It orchestrates five specialized LLM agents that process information in a logical flow, ensuring each stage builds upon the data extracted or generated by the previous one.

## 🛠️ Tech Stack
* **Language:** Python
* **AI Model:** `gemini-2.5-flash`
* **Framework:** `google.adk.agents` (`LlmAgent`, `SequentialAgent`)
* **Data Handling:** Strict JSON schema for ATS-compliant parsing

## 📂 File Structure
* **`prompt_for_resume.py`**: Contains the complex system instructions and the strict JSON schema for extracting personal info, work history, and categorized skills.
* **`agent.py`**: The core logic where individual agents are defined and chained together into the `SequentialAgent`.
* **`__init__.py`**: Handles package initialization and imports.

## 🤖 The Agent Pipeline
The `job_finder_agent` executes the following sequence:

| Step | Agent Name | Responsibility | Output Key |
| :--- | :--- | :--- | :--- |
| 1 | **Resume_Extraction_Agent** | Parses raw resume text into structured JSON. | `extracted_resume_data` |
| 2 | **Role_Suggestion_Agent** | Suggests 2-4 job roles based on skills. | `suggested_roles` |
| 3 | **Job_Search_Agent** | Finds openings matching the suggested roles. | `recommended_jobs` |
| 4 | **Interview_Questions_Agent** | Generates technical and behavioral questions. | `interview_questions` |
| 5 | **Interview_Tips_Agent** | Provides personalized coaching and strategy. | `interview_tips` |

## 📋 Extraction Schema
The extraction agent is programmed to strictly adhere to a JSON schema. If information is missing, it defaults to `null` rather than placeholders like "N/A".

**Key Categories Extracted:**
* **Personal Info**: Name, contact, and social URLs (LinkedIn, GitHub).
* **Work Experience**: Roles, companies, dates, and technologies used.
* **Skills**: Categorized into Programming, Frameworks, Cloud, and Databases.
* **Projects & Certifications**: Descriptions, URLs, and issuing organizations.
